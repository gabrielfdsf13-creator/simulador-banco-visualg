# simulador-banco-visualg
Simulador de sistema bancário desenvolvido no VisuAlg para o projeto de Algoritimos e Estrutura de Dados. O programa permite cadastro, consultar saldo, realizar depósitos, efetuar saques e ver histórico de operações. O objetivo foi aplicar conceitos de algoritmos, variáveis, condições e repetições para resolver um problema prático do dia a dia.
